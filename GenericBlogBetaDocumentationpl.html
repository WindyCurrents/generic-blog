Dokumentacja projektu generic-blog-beta

Pozycja 1. connect.php

Podtytuł a .: Pierwsze 4 części kodu

skrypt ustanawia 4 zmienne połączenia z bazą danych użytkowników bloga, $ host = "localhost", $ user = "root", $ pass = "", $ dbname = "genericblogbetadatabase"

Subb.: Reszta kodu

używając zmiennych w pierwszych wierszach, skrypt próbuje połączyć się z bazą danych

Pozycja 2. genericblogbetadatabase.sql

jest to baza danych użytkowników z tabelą użytkowników zawierającą kolumny identyfikatora, adresu e-mail, nazwy użytkownika i hasła. id jest kluczem podstawowym

Pozycja 3. index.php

jest to niestylizowana strona informująca użytkownika, że jest to niedokończona wersja projektu ogólnego bloga. znajduje się link do logowania do serwisu lub rejestracji. może również pokazać łącze wylogowania, jeśli użytkownik jest teraz zalogowany.

Pozycja 4. login.php

och, wow, to jest strona logowania

Pozycja 5. logout.php

usuwa dane związane z sesją i powraca do indeksu

Element 6. register.php

wow rzeczywista strona rejestracji, ma pewne filtry, takie jak 3-8 znaków w nazwie użytkownika? lub co najmniej 8 znaków w haśle

Element 7. utils.php

Obecnie ma tylko funkcje związane z logowaniem, później będzie można wstawiać i wyświetlać posty według ich identyfikatora

Element 8. insertform.php

Prosty formularz z prośbą o podanie tytułu i treści nowego posta.

Element 9. insertscript.php

Pobiera wprowadzone dane oraz identyfikator użytkownika i wysyła je do bazy danych.

Element 10. searchform.php

Prosty formularz z prośbą o podanie identyfikatora posta.

Element 11. searchscript.php

Wyszukuje wpis z identyfikatorem podanym w formularzu.
