Documentation of the generic-blog-beta project

Point 1. connecttodb.php

Subheading a .: First 4 parts of the code

the script establishes 4 connection variables to the blog user database, $ host = "localhost", $ user = "root", $ pass = "", $ dbname = "genericblogbetadatabase"

Subb.: Rest of code

using the variables in the first lines, the script tries to connect to the database

Item 2. genericblogbetadatabase.sql

it is an empty user database with a userdata table containing the UserID, Username and Password columns. UserID is the root key

Item 3 index.php

it is a nonstylized page informing the user that this is an unfinished version of the generic-blog project. there is a link to login to the admin panel.

Item 4. loginmanagescript.php

script consists of 4 functions for handling user login. getuserid and getusername both return the isloggedon function with the user id or username. isloggedon returns $ _SESSION to the logged in user id. logon takes data from the form at loginpage.php and tries to find a matching record in the database to log the user in.

Last point. loginpage.php

it is a simple login form which only contains 2 fields and a login button.
