Documentation of the generic-blog project

Point 1. 127_0_0_1.sql

This is the data base file you import to the server to have a working database to this project.

Point 2. connect.php

Contains credentials to the database server and attempts a connection.

Point 3. index.php

The main page of this project. Shows some options only to logged in users and has scripts regarding messages about posting or deleting.

Point 4. insertform.php

A simple form with 2 fields that gets the data to insertscript.php.

Point 5. insertscript.php

Gets data from insertform.php and adds it to the posts table of the database. Also sets a session value later read by index.php to display a message.

Point 6. listpostids.php

Checks your user ID against all posts' poster IDs and displays IDs of the posts this matches to.

Point 7. login.php

A login form which also contains login scripts.

Point 8. logout.php

Unsets everything session-related and returns you to index.php.

Point 9. postdeleteform.php

A one-field form that sends its result to postdeletescript.php.

Point 10. postdeletescript.php

Ensures you're deleting a post of your own and if successful, sets a session value later read by index.php to display a message.

Point 11. register.php

A normal registration page with 2 filters: User name must be 3-16 characters, alphanumeric characters and underlines only, and password must be at least 8 characters.

Point 12. searchform.php

postdeleteform.php, except sends it to searchscript.php.

Point 13. searchscript.php

Attempts finding a post of the provided ID and displaying it.

Point 14. utils.php

Login and register-related functions.

Point 15. editform.php

Asks for the ID of the post you'll edit, its new title and contents. Sends everything to editscript.php.

Point 16. editscript.php

Ensures the edited post belongs to you and proceeds to update it. Also sets a session value later read by index.php to display a message.
