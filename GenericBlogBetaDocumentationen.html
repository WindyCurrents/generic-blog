Documentation of the generic-blog-beta project

Item 1. connect.php

Subheading a .: First 4 parts of the code

the script establishes 4 connection variables to the blog user database, $ host = "localhost", $ user = "root", $ pass = "", $ dbname = "genericblogbetadatabase"

Subb.: Rest of code

using the variables in the first lines, the script tries to connect to the database

Item 2. genericblogbetadatabase.sql

it is an user database with a users table containing the id, email, username and password columns. id is the primary key

Item 3. index.php

it is a nonstylized page informing the user that this is an unfinished version of the generic-blog project. there is a link to login to the site or register. could also show a logout link if user is now logged in.

Item 4. login.php

oh wow it is a login page

Item 5. logout.php

unsets session related data and returns you to index

Item 6. register.php

wow an actual registration page, has some filters like 3-8 characters in username?? or at least 8 characters in password

Last item. utils.php

Currently only has login-related functions, will later on be able to insert and show posts by their ID
