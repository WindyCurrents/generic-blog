Project "Generic Blog" - Documentation

1. Introduction

This is the documentation for the project "Generic Blog". The repository for this project dates back to September 7th, 2022.

2. Installation

The project was only tested to work with a XAMPP Apache+MySQL environment. It goes as follows:

  1. Download and install XAMPP from https://www.apachefriends.org/index.html
  2. Clone the repository to the htdocs folder of your XAMPP installation
  3. Start the Apache and MySQL services from the XAMPP Control Panel
  4. Open the phpMyAdmin page from the XAMPP Control Panel
  5. Import the 127_0_0_1.sql file directly to the server. It's a skeleton of the database, with no actual data in it.
  6. Open the project in your web browser by going to "http://localhost/generic-blog" (or however you named the folder with project files)

3. Usage

The project is a simple blog, with the following features:

  • Post creation, viewing, editing and deleting
  • Login and registration

4. Explanation of all files

The project is divided into the following files:

  • 127_0_0_1.sql - The skeleton of the database.
  • connect.php - The file that connects to the database.
  • index.php - The main page of the blog, with functions at disposal being displayed.
  • listpostids.php - A simple page that displays the IDs of all posts made by the currently logged in user.
  • login.php - The login page.
  • logout.php - A very short PHP script that unsets everything related to the session and redirects the user back to index.php.
  • postaddform.php - The page where the user can create a new post.
  • postaddscript.php - The script that adds a new post to the database.
  • postdeleteform.php - The page where the user can delete a post.
  • postdeletescript.php - The script that deletes a post from the database.
  • posteditform.php - The page where the user can edit a post.
  • posteditscript.php - The script that edits a post in the database.
  • register.php - The registration page.
  • searchform.php - The page where the user can search for a post based on its ID.
  • searchresults.php - The script that searches for a post based on its ID, and displays it to the user.
  • styles.css - The stylesheet for the project.
  • utils.php - The center for PHP functions used by the project.

5. Conclusion

This project is a simple blog, with the main goal of demonstrating the use of PHP and MySQL in a web application.
